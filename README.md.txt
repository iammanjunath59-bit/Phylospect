# Phylospect

Phylospect is a spectral operator–based framework for detecting branch-coherent episodic selection in codon evolution.

## Requirements
Python ≥ 3.9  
numpy  
scipy  
pandas  
matplotlib  

## Run complete analysis

python run_phylospect_pipeline.py

This generates:

data/results_phylospect.csv

## Reproducibility

All results in the manuscript can be regenerated from these scripts.
