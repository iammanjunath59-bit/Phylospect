# PHYLOSPECT

**Phylogenetic Spectral Operator-based Codon Testing**

A likelihood-free, operator-based framework for detecting episodic positive
selection at individual branches of a phylogenetic tree.

PHYLOSPECT estimates branch-specific codon substitution rate matrices via
stochastic mapping, computes the **Nonsynonymous Operator Excess (NOE)** — the
signed mean deviation from a pooled background across nonsynonymous
single-nucleotide codon pairs — and evaluates it against a parametric bootstrap
null. It is designed as a fast, calibrated pre-screening tool for genome-scale
positive-selection surveys, complementing rather than replacing likelihood-based
confirmatory tests such as PAML branch-site Model A.

---

## Key features

- **Calibrated** — parametric bootstrap null maintains FPR ≈ 0.05 across tree
  configurations and alignment lengths
- **MNM-robust** — 3.3× lower false positive rate than PAML under
  multinucleotide mutation contamination (a documented mammalian model
  violation)
- **Fast** — 5–10× faster per branch than PAML; cost scales with alignment
  length, not tree size
- **Complementary** — detects branch-coherent distributed selection; PAML
  detects site-localised selection; both together cover more biological signal

## Quickstart

```python
import numpy as np
from phylospect.gy94 import build_gy94_generator, uniform_codon_frequencies
from phylospect.pipeline import run_phylospect

# Load your alignment
def read_fasta(path):
    aln = {}; name = None; chunks = []
    with open(path) as fh:
        for line in fh:
            line = line.strip()
            if not line: continue
            if line.startswith('>'):
                if name: aln[name] = ''.join(chunks)
                name = line[1:].split()[0]; chunks = []
            else: chunks.append(line)
        if name: aln[name] = ''.join(chunks)
    return aln

aln    = read_fasta('vignette/example_alignment.fasta')
newick = "((Taxon_A:0.08,Taxon_B:0.08)AB:0.04,(Taxon_C:0.08,Taxon_D:0.08)CD:0.04);"

# Build GY94 background model
pi = uniform_codon_frequencies()
Q0 = build_gy94_generator(omega=1.0, kappa=2.0, pi=pi, scale=True)

# Run PHYLOSPECT
results = run_phylospect(newick, aln, Q0,
                         n_perm=100,
                         rng=np.random.default_rng(42))

# Report
for branch, r in sorted(results.items()):
    sig = ' *' if r['pvalue'] <= 0.05 else ''
    print(f"{branch:<12} NOE={r['NOE']:+.4f}  p={r['pvalue']:.3f}{sig}")
```

## Reproducing published results

Each experiment script runs with fixed seeds and produces results consistent
with the manuscript. From the repo root:

```bash
python experiments/01_calibration.py        # ~5 min
python experiments/02_power.py              # ~6 h
python experiments/03_alignment_length.py   # ~2 h
python experiments/04_paml_comparison.py    # requires codeml
python experiments/05_runtime.py            # requires codeml
python experiments/06_branch_length.py      # ~1 h
python experiments/07_basis_sensitivity.py  # ~8 h
python experiments/08_misspecification.py   # requires codeml
python experiments/09_empirical_v4.py data/empirical/lysozyme.fasta \
    data/empirical/lysozyme.tre lysozyme data/empirical/lysozyme_clades.txt
```
